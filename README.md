# 축제로 🎪

대한민국 전국 시군구의 **사계절 축제**를 지도에서 한눈에 보고, **날씨**와 **길찾기**까지 확인하는 웹사이트입니다.

## 주요 기능

- 🗺️ 전국 지도에 축제 위치 표시 (마커 클릭 → 축제명·기간 팝업)
- 🌸 계절(봄·여름·가을·겨울) / 지역(전국·서울·경기인천·강원·충청·전라·경상·제주) 선택
- 🏷️ 진행중 / 예정 / 종료 상태 배지가 붙은 축제 카드 목록
- 🌤️ 축제 상세: 오늘~3일 날씨 (Open-Meteo)
- 🧭 카카오맵 길찾기 (내 위치에서 출발)
- 📱 모바일 우선 반응형 + 계절별 테마 색상

## 기술 스택

| 구분 | 사용 기술 |
| --- | --- |
| 프레임워크 | Next.js (App Router) + React |
| 지도 | Leaflet + OpenStreetMap (react-leaflet) |
| 날씨 | Open-Meteo API (무료, 키 불필요) |
| 길찾기 | 카카오맵 링크 (키 불필요) |
| 축제 데이터 | 한국관광공사 TourAPI (searchFestival2) + 내장 샘플 데이터 |

## 로컬에서 실행하기

```bash
npm install     # 처음 한 번만
npm run dev     # 개발 서버 실행
```

실행 후 브라우저에서 **http://localhost:3000** 접속.

## 실데이터(TourAPI)로 전환하기

지금은 키가 없어도 **내장 샘플 축제 데이터**로 동작합니다.
나중에 [공공데이터포털](https://www.data.go.kr)에서 한국관광공사 TourAPI 키를 발급받으면:

1. `.env.local.example` 파일을 복사해 `.env.local` 로 이름을 바꾸고
2. 파일 안 `TOUR_API_KEY=` 뒤에 발급받은 키를 붙여넣은 뒤
3. 개발 서버를 다시 실행하면 실데이터로 자동 전환됩니다.

> `.env.local` 은 `.gitignore`에 등록되어 있어 GitHub에 올라가지 않습니다(키 보호).
