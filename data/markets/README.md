# 전통시장(장터·야시장) 데이터 안내 (기획자용)

지도 필터의 "🏪 장터·야시장" 칩을 켜면 나오는 시장 목록입니다. 공공데이터
전통시장 표준데이터 API가 있으나 **정부 게이트웨이가 느리고 별도 활용신청(키 등록)이
필요**해, 우선 이 파일(`markets.json`)에 직접 등록해 시작합니다.

## 시장 추가하는 법
`markets.json`의 `markets` 배열에 아래 형식으로 한 줄 추가하면 됩니다.

### 5일장(정기시장)
```json
{ "id": "mkt-예시", "name": "○○5일장", "marketType": "5일장",
  "sido": "경상북도", "sigungu": "안동시", "region": "gyeongsang",
  "lat": 36.57, "lng": 128.72, "openDays": [2, 7], "hours": "", "addr": "..." }
```
- `openDays`: **장서는 날의 끝자리**. 예) 2·7일장 → `[2, 7]`, 4·9일장 → `[4, 9]`, 5·10일장 → `[5, 0]`
- 이러면 "다음 장날: 8/3(월)"이 자동 계산되고, 오늘이 장날이면 "오늘 장서는 날!" 배지가 뜹니다.

### 야시장(상설)
```json
{ "id": "night-예시", "name": "○○야시장", "marketType": "야시장",
  "sido": "부산광역시", "sigungu": "중구", "region": "gyeongsang",
  "lat": 35.09, "lng": 129.03, "openDays": [], "hours": "금·토 18:00~24:00", "addr": "..." }
```
- `openDays`는 빈 배열 `[]`, 대신 `hours`에 운영시간을 적습니다.

### region 코드(권역)
seoul(서울) · gyeonggi(경기·인천) · gangwon(강원) · chungcheong(충청·대전·세종) ·
jeolla(전라·광주) · gyeongsang(경상·부산·대구·울산) · jeju(제주)

### 좌표(lat/lng)
지도·PC 지도에서 시장 위치를 찾아 위도(lat)·경도(lng)를 넣습니다(근사값도 괜찮음).

## 공공데이터 API 자동 보강(선택)
전국전통시장표준데이터(data.go.kr 15012894) 활용신청 후 키를 발급받으면,
`src/lib/markets.js`의 커넥터를 켜서 5일장을 자동으로 보강할 수 있습니다(현재 기본 꺼짐).
